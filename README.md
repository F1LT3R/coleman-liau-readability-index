# Coleman Liau Readability Index

Calculates the Coleman Liau readability index of a peice of text.

## Installation

Checkout the code

```shell
git clone git@github.com:f1lt3r/coleman-liau-readability-index.git
```

Install packages

```shell
cd coleman-liau-readability-index

yarn install
```

## Usage

Run tests

```shell
yarn test
```
